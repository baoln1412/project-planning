## Summary

A web application marketplace where users can purchase premium life/habit tracker dashboards. The unique value proposition is that the dashboards connect directly to the user's own Google Sheets, which act as their personal database. The platform also includes an admin toolset powered by Antigravity AI to rapidly generate new dashboard products from mockups and descriptions.

## Problem Statement

Users want beautiful, functional dashboards to track their lives (reading, habits, goals), but they don't want their personal tracking data locked into a proprietary system. Standard spreadsheet templates are ugly, while dedicated SaaS apps are rigid and own the data. 

As a creator, building new dashboards from scratch takes too much time.

## Target Users

- **Consumers**: People who want premium tracking dashboards but prefer to own their data via Google Sheets.
- **Admin/Creator**: The workspace owner (you) — who needs to rapidly build and sell new dashboards using AI assistance.

## Discovery Answers

1. **🎯 North Star**: To generate passive income by selling premium life tracker dashboards to interested users.
2. **🔌 Integrations**: Google Sheets API (user data), Antigravity AI (dashboard generation), Stripe (assumed for payments).
3. **💾 Source of Truth**: User's Google Sheets (tracking data) + Web App Database (user accounts, purchases, dashboard catalog).
4. **📦 Delivery Payload**: A Web Application.
5. **🚫 Behavioral Rules**: Do not store user tracking data on our servers, only read from their Google Sheets.

## Key Features

- **Storefront & Product Pages**: Browse available dashboards (e.g., Reading Tracker, Habit Tracker) and instructional home page.
- **Continuous AI Expansion Architecture**: The core system is designed to seamlessly integrate new dashboard pages and components strictly generated by Antigravity AI based on creator prompts and mockups. Ensure Antigravity researches Tremor components and uses the `_design/` folder to adhere to brand guidelines.
- **Master Data Sheet Cloning**: Every dashboard has a corresponding Google Sheet template. Users clone the master sheet, use it as their database, and connect it to the web app dashboard.
- **Read & Write Integration**: The web app reads from and writes to the user's connected Google Sheet (e.g., creating plans, setting goals directly in the UI).
- **Secure Manual Approval Flow**: To mitigate transaction security risks and prevent data leaks, purchases are verified manually by an admin account before granting users access to specific dashboards.
- **Dedicated Design System & Brand Asset Library**: Centralized storage (`_design/`) for logos, styling rules, URL preferences, and UI component summaries. Antigravity IDE uses this as a source of truth when generating new templates to ensure a consistent, branded look and feel.
